# 🏠 Buscando Techo | Tu Hogar en La Habana

Sitio web de agencia inmobiliaria especializado en propiedades en La Habana, Cuba.

![Buscando Techo](https://z-cdn-media.chatglm.cn/files/3110672a-28e2-4457-b485-06747a861b54.jpg)

## ✨ Características

- 🏡 Catálogo de propiedades (casas y apartamentos)
- 🔍 Filtros de búsqueda por precio, municipio y texto
- 🗺️ Mapa interactivo de municipios de La Habana
- 📱 Diseño responsive (mobile-first)
- 🎨 Animaciones suaves y modernas
- 📞 Múltiples canales de contacto (Facebook, Telegram)
- ➕ Panel para agregar nuevas propiedades

## 🚀 Despliegue en GitHub Pages

1. Crea un nuevo repositorio en GitHub
2. Sube los archivos del proyecto
3. Ve a **Settings > Pages**
4. En **Source**, selecciona `main branch` y guarda
5. Tu sitio estará disponible en `https://tu-usuario.github.io/tu-repositorio`
